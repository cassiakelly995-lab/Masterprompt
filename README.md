## 📌 O que é este projeto?

Uma **biblioteca interativa de prompts profissionais** prontos para uso no ChatGPT, Claude e Gemini.

Cada prompt foi desenvolvido com:
- Estrutura profissional com variáveis editáveis `[entre colchetes]`
- Restrições explícitas para evitar respostas vagas ou inventadas
- Instruções de entrega organizadas e acionáveis
- Botão de cópia direta para a área de transferência

## 🛡️ Destaque — Prompt Anti-Alucinação v3.0

O prompt mais importante da biblioteca. Um **framework de controle de qualidade** que pode ser colado no início de qualquer outro prompt para eliminar:

- Estatísticas inventadas
- Citações não verificadas
- Respostas vagas usadas para esconder ignorância
- Datas e preços desatualizados apresentados como fatos

Funciona como um **bloco de instruções universal** — compatível com qualquer IA e qualquer tarefa.

---

## ⚙️ Como usar

1. Acesse a [demo no GitHub Pages](https://cassiakelly995-lab.github.io/Masterprompt/)
2. Filtre por categoria ou navegue pelos prompts
3. Clique em **Ver Prompt** para expandir o conteúdo completo
4. Substitua os campos `[entre colchetes]` com suas informações
5. Clique em **Copiar** e cole direto no ChatGPT, Claude ou Gemini
